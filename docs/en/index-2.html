2. Installation

2.1 How to obtain QLocate

See http://www.geocities.com/rebusnet/qlocate/index.html

2.2 Requirements 

- QT library 2.3.0 or superior,
- slocate package,
- rpm command (optional),
- file command (optional).

Qlocate is tested with slocate 2.5 and more.
RPM part is tested with rpm 4.0.3 and more.
file part is tested with file 3.36 and more.

2.3 Compilation and installation

In order to compile and install QLocate on your system, type the following in the base directory of the QLocate distribution: 

% ./configure
% make
% make install

Since QLocate uses 

autoconf
you should have not trouble compiling it. Should you run into problems please report them to the the author at Steeve Beaudin
